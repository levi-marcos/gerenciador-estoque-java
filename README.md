# 📦 Gerenciador de Estoque Java

Um sistema de console simples para praticar meus conhecimentos, desenvolvido em Java para gerenciar o cadastro de produtos e preços, utilizando coleções dinâmicas para armazenamento.

## 🚀 Funcionalidades

* **Cadastro de Produtos:** Permite inserir o nome e o preço de diversos itens.
* **Armazenamento:** Utiliza `HashMap` para garantir que cada produto seja único e tenha acesso rápido.
* **Relatório Automático:** Exibe uma lista detalhada de todos os itens cadastrados ao final da execução.
* **Cálculo de Total:** Soma automaticamente o valor de todo o inventário.

## 🛠️ Tecnologias Utilizadas

* **Java**
* **Scanner** Para interação via console.
* **Collections (Map/HashMap):** Para gerenciamento de dados em memória.
